## Quick Start

```bash
# Clone
git clone https://github.com/aro-brez/8owls-app.git
cd 8owls-app

# Backend
cd server
pip install -r requirements.txt
cp .env.example .env  # Add your API keys
uvicorn app:app --reload

# Frontend
cd ../app
npm install
npm run dev
```
